# AI Chat Navigator Extension

这是一款为 AI 对话页面提供右侧浮动导航条的浏览器扩展，帮助在长对话中快速定位用户提问并跳转。当前适配 chatgpt.com、chat.openai.com、cc01.plusai.io 与 gemini.google.com，后续计划扩展更多站点。

做这个插件的契机
长对话里回看某个提问需要频繁滚动，效率低。左侧历史列表无法解决同一对话内部的快速定位需求。

功能要点
- 识别对话中的用户提问，按出现顺序生成导航条目
- 标题显示提问内容，最多两行，超出内容在行尾截断
- 预览显示对应的 AI 回复，单行展示并省略超长内容
- 点击导航项后，页面平滑滚动到该提问位置并短暂高亮
- SPA 路由切换与内容增量变化都会触发重建与刷新
- 面板支持折叠为右侧圆形按钮，按钮可拖动并记忆位置

**安装与使用**
1. 打开 `chrome://extensions` 并启用开发者模式
2. 点击“加载已解压的扩展程序”，选择此仓库目录
3. 打开支持站点的对话页面，右侧浮动面板会自动出现

**目录结构**
- `manifest.json` 扩展清单与注入规则
- `content/` 内容脚本模块
- `content.js` 启动入口

**配置与隐私**
仅声明必要的站点权限，内容脚本不包含外部网络请求。对话内容只用于生成本地导航列表。

**维护建议**
新增站点时请在 `content/adapter.js` 里扩展适配器，并在 `manifest.json` 增加 host 权限。
